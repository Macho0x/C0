# Goop 1.7 Go struct FFI (design note)

**Date:** 2026-07-12  
**Status:** Implemented in 1.7.0

## Summary

Imported Go structs (`import go { type T }`) are classified via gosig
`LookupType`. Structs get field schemas; interfaces keep `Interface: true`.

Record literals type as `T` when the expected type is `T` or `T ptr`
(call args and return annotations). `ptr_of { … }` emits `&pkg.T{…}`.
Field names match Go exports case-insensitively. Omitted fields are zero.
Interface fields accept Go-assignable named types (`Level` → `Leveler`).

## Non-goals (unchanged)

Auto-import without `type` decls; unexported fields; `{ s with … }` updates;
generic Go structs.
